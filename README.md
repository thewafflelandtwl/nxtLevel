# NXT Level - Branding & Digital Growth Agency

A modern, sophisticated website for NXT Level, a premium branding and digital growth company specializing in brand identity, website design, video production, and digital marketing.

## Features

### Design Aesthetics
- **Bold Editorial Design**: Inspired by the company's gradient-bordered logo with modern typography
- **Distinctive Typography**: Bebas Neue for display headings, Outfit for body text
- **Gradient Accents**: Purple-to-blue gradient theme matching the brand identity
- **Dark Theme**: Sophisticated dark background with subtle grid animations

### Animations & Interactions
- Smooth scroll-triggered animations using CSS animation-timeline (with fallback)
- Parallax effects on hero section
- Magnetic hover effects on primary buttons
- 3D tilt effect on service cards
- Animated counters in stats section
- Staggered reveal animations for hero text
- Floating decoration elements

### Sections
1. **Navigation**: Fixed header with smooth scroll links and CTA button
2. **Hero Section**: Bold, large typography with animated word reveals
3. **Services**: 6 service cards featuring:
   - Brand Identity
   - Website Design
   - Video Production
   - Social Media Marketing
   - SEO Services
   - Content Creation
4. **Portfolio**: Link to Behance portfolio showcase
5. **Stats**: Animated statistics counters
6. **CTA Section**: Contact section with WhatsApp and Instagram links
7. **Footer**: Complete site map with social links

### Technical Features
- Fully responsive design (mobile, tablet, desktop)
- Modern CSS with custom properties
- Intersection Observer for performance-optimized animations
- Smooth scrolling behavior
- SEO-friendly semantic HTML
- Accessibility considerations
- Optimized performance

## How to Use

1. **Local Development**:
   Simply open `index.html` in a modern web browser (Chrome, Firefox, Safari, Edge)

2. **Deploy to Web**:
   - Upload all files to your web hosting service
   - Ensure all files are in the same directory
   - The website will be accessible via your domain

3. **Customization**:
   - Update content in `index.html`
   - Modify colors in CSS variables at the top of `styles.css`
   - Add portfolio images by replacing the `.image-placeholder` element
